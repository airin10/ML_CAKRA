ML CAKRA - Marketing Analytics Platform
Aplikasi web canggih untuk evaluasi model machine learning regression dengan fokus pada analisis performa pemasaran dan media channels. Platform ini dirancang khusus untuk membantu tim marketing menganalisis efektivitas berbagai channel media dalam mendorong penjualan.

Fitur Utama :
- Analisis Regresi Komprehensif
  Multiple Regression Algorithms: Linear, Ridge, Lasso, dan Random Forest Regression

- Automated Model Evaluation
  Evaluasi otomatis dengan berbagai metrik performa

- Cross-validation Built-in
  Validasi model yang robust untuk hasil yang dapat diandalkan

Advanced Data Preprocessing
Smart Data Cleaning:
- Automatic handling missing values
- Konversi tipe data numerik (termasuk format currency)
- Deteksi dan treatment outlier otomatis
- Feature Engineering Cerdas:
- Aggregasi media channels otomatis
- Log transformation untuk stabilitas variance
- Winsorization pada target variable
- Automated Feature Selection: Pemilihan fitur optimal berdasarkan business rules

Visualisasi Interaktif
- Actual vs Predicted Scatter Plot: Visualisasi akurasi model
- Residual Analysis: Diagnostik model yang komprehensif
- Feature Importance Charts: Identifikasi channel media paling berpengaruh
- Error Distribution Analysis: Pemahaman mendalam tentang error pattern
- Confusion Matrix Binned: Klasifikasi performa berdasarkan kategori sales

Marketing-Specific Analytics
- Media Channel Aggregation: Grup otomatis channel media yang serupa
- ROI Estimation: Analisis return on investment per channel
- Performance Segmentation: Kategorisasi Low/Medium/High sales
- Channel Effectiveness Ranking: Ranking media channels berdasarkan impact

Model yang Didukung
1. Linear Regression
- Model dasar untuk hubungan linear
- Interpretasi koefisien yang straightforward
- Cocok untuk data dengan hubungan linear yang kuat

2. Ridge Regression
- Regularisasi L2 untuk menghindari overfitting
- Stabil dengan multicollinearity
- Ideal untuk dataset dengan banyak fitur berkorelasi

3. Lasso Regression
- Regularisasi L1 dengan feature selection built-in
- Automatis seleksi fitur paling penting
- Cocok untuk sparse datasets

4. Random Forest Regressor
- Ensemble method yang robust terhadap outlier
- Capture non-linear relationships
- Feature importance yang reliable
